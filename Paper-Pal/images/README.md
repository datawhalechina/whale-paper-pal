# Images

This directory contains images used in the project documentation.

## Structure

- `screenshots/` - Application screenshots for README
- `icons/` - Project icons and logos
- `demo/` - Demo images and GIFs

## Usage

Images in this directory are referenced in:
- `README.md` - Main project documentation
- `README_CN.md` - Chinese project documentation

## Guidelines

- Use descriptive filenames
- Optimize images for web (reasonable file sizes)
- Prefer PNG for screenshots, JPG for photos
- Consider using GIFs for animated demonstrations
